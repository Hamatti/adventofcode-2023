## Jupyter Notebook

1. Activate virtualenv:

```
source venv/bin/activate
```

(1.5 Install Jupyter Notebook

```
pip install notebook
```

)

2. Run Jupyter Notebook server

```
jupyter notebook
```

3. Convert to HTML

```
jupyter nbconvert to=html [notebook]
```
